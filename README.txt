Indicazioni per la consultazione dei file aggiuntivi:

- La cartella <File aggiuntivi/poeti> contiene tutti i file txt delle poesie catalogati per autore.
- Il file webscraper3.py è uno script python che, preso in input un url web, scansiona la pagina html
  alla ricerca del contenuto di testo da copiare ed incollare in un apposito file .txt.
  Lo script è modificabile ridefinendo il path di salvataggio del file ed il nome dell'autore.
- Il file words_to_be_deleted.txt contiene le parole e numeri da preprocessare ed eliminare
  dal dataset finale. Viene fornito al modello in fase di text preprocessing.
- Il file report.pdf è il report Orange creato in fase di salvataggio del modello.
